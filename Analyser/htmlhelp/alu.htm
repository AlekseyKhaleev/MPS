3.12. АЛУ с полным управлением вычислениями

 

Данное АЛУ состоит из двух регистров (RGA и RGB) и накапливающего сумматора (SM).

Регистры имеют разрядность 8 (7 разрядов данных + 1 разряд знака). Над данными в регистрах могут выполняться операции инверсии и сдвига. Обмен данными с другими устройствами осуществляется только через регистры.

 

Накапливающий сумматор имеет разрядность 16 (14 разрядов данных + 1 разряд знака + 1 разряд переполнения). Сумматор поделён на две части: старшую (D13-D7) и младшую (D6-D0). Все разряды сумматора используются при умножении/делении 7-ми разрядных чисел со знаком. При сложении/вычитании чисел указанного вида необходимо использовать старшую часть сумматора. Входы сумматора являются счётными (при каждой записи в сумматор его содержимое складывается с записываемыми в него данными). Кроме того, в сумматоре реализуются операции инверсии и сдвига через знаковый разряд. Циклический сдвиг может быть реализован установкой перемычек между разрядами SM из групп D15-D13(выходы) и D7, D0 (входы).

 

Регистры и сумматор подключены к общей внутренней шине, которая, кроме передачи данных, играет роль монтажного ИЛИ.

 

На УГО данного АЛУ отображена схема передачи данных между регистрами и сумматором. Сигнальные лампочки на ШФ схемы соответствуют их CS и показывают путь прохождения данных через схему. Разряды регистров и сумматора снабжены индикацией.

 

Возможности АЛУ

1.      Обеспечивается базис ИЛИ-НЕ для выполнения логических операций над 8-ми разрядными операндами.

2.      Сдвиг поля данных, сдвиг данных через знаковый разряд, циклический и нециклический сдвиги.

3.      Выполнение умножения 8-ми разрядных чисел со знаком по четырём схемам умножения.

4.      Сложение/вычитание.

5.      Образование прямого и дополнительного кода.

6.      Деление 8-ми разрядных чисел со знаком

 

УГО АЛУ.

 

 

Управляющие линии регистра RG(RGA или RGB)

Inv I/O

При вводе/выводе на внутреннюю шину данных используются инверсные входы/выходы триггеров данных регистра  RG

D6 ONLY

При вводе/выводе на внутреннюю шину данных на каждый разряд этой шины выдается значение триггера D6 регистра RG

D0 ONLY

При вводе/выводе на внутреннюю шину данных на каждый разряд этой шины выдается значение триггера D0 регистра RG

Mov In

Приём разрядов данных в RG с внутренней шины

Mov Out

Выдача разрядов данных RG на внутреннюю шину

RD

Выдача всех разрядов RG на внешнюю шину

WR

Приём всех разрядов RG с внешней шины

MovS Out

Выдача разряда знака (D7) RG на внутреннюю шину

MovS In

Приём разряда знака в RG с внутренней шины

Set S 0

Установить разряд знака RG в 0

Inv S

Инверсия знакового разряда RG

Inv DT

Инверсия разрядов данных RG

DT>>1

Сдвиг данных RG на 1 разряд вправо

DT<<1

Сдвиг данных RG на 1 разряд влево

CLR DT

Очистка разрядов данных RG

 

Данные поступают в регистр по раздельным входам (содержимое регистра перезаписывается).

 

Управляющие линии накапливающего сумматора SM

Inv I/O

При вводе/выводе на внутреннюю шину данных используются инверсные входы/выходы триггеров данных сумматора SM

Mov OutL

Выдача младших разрядов данных (D6-D0)SM на внутреннюю шину

Mov OutH

Выдача старших разрядов данных (D13-D6)SM на внутреннюю шину

MovS Out

Выдача разряда знака (D14) SM на внутреннюю шину

RD

Выдача разрядов D15-D13 SM на внешнюю шину

WR

Приём разрядов D7 и D0 SM с внешней шины

Mov InL

Приём  младших разрядов данных (D6-D0) SM с внутренней шины

Mov InH

Приём  старших разрядов данных (D13-D6) SM с внутренней шины

MovS In

Приём разряда знака (D14) SM с внутренней шины

Set S 0

Установить разряд знака SM в 0

Inv S

Инверсия знакового разряда SM

Inv DT

Инверсия разрядов данных SM

DT>>1

Сдвиг всего содержимого SM на 1 разряд вправо

DT<<1

Сдвиг всего содержимого SM на 1 разряд влево

S+DT

Прибавить к данным SM знаковый разряд

CLR DT

Очистка разрядов данных SM

 

D0-D6 – младшие разряды сумматора

D7-D13 – старшие разряды сумматора

D14 – знаковый разряд сумматора

D15 – флаг переполнения сумматора

Выходы D15-D13 и входы D7 и D0 используются для создания перемычек в Алу (например при реализации циклического сдвига).

Данные поступают в сумматор по счётным входам (новые данные складываются с содержимым сумматора).
